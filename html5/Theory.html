Q1: What are the new tags added in HTML5?

Ans:
HTML5 Tags:

HTML5 introduced several new tags to improve semantics and provide more structure to web documents. Some of the key new tags added in HTML5 include:

1. header tag: Represents introductory content, typically containing navigation links, logos, and headings.


2. footer tag: Defines the footer of a section or a page, usually containing copyright information, contact details, or other metadata..


3. nav tag: Specifies a section of navigation links, such as menus or navigation bars..


4. article tag: Represents an independent piece of content that makes sense on its own, such as blog posts, news articles, or forum posts.


5. section tag: Defines a generic section of a document, often used to group related content together.


6. aside tag: Represents content that is tangentially related to the content around it, such as sidebars or pull quotes..


7. main tag:Specifies the main content of a document, excluding any header, footer, or sidebar content..


8. figure and figcaption tag: Used together to represent self-contained content, such as images, videos, diagrams, or code snippets, along with a caption..


9. datalist tag: Provides a list of predefined options for input controls like input or textarea.


10. progress tag: Represents the progress of a task, such as a file download or form submission.


11. meter tag: Represents a scalar measurement within a known range, such as disk usage, ratings, or voting results.


12. details and summary tag:Used together to create a collapsible section that can be toggled open or closed, useful for hiding secondary information until it's needed.

Q2: How to embed audio and video in a webpage?

Ans:
To embed audio and video in a webpage, HTML5 provides the audio and video tags. Here's how you can use them::


The audio tag is used to embed audio content. The controls attribute adds standard audio controls (play, pause, volume) to the audio player. The source tag is used to specify multiple sources for the audio, allowing the browser to choose the most suitable format based on its capabilities. You can provide different formats like MP3, OGG, or WAV. The src attribute specifies the URL of the audio file. The type attribute specifies the MIME type of the audio file.


The video tag is used to embed video content. The controls attribute adds standard video controls (play, pause, volume, fullscreen) to the video player. The width and height attributes specify the dimensions of the video player. Similar to audio, you can use multiple source tags to provide different video formats for compatibility across browsers and devices. The src attribute specifies the URL of the video file. The type attribute specifies the MIME type of the video file.

Q4:Canvas and SVG tags

1.Canvas:

canvas is an HTML5 element that provides a drawing space for graphics using JavaScript.
It's essentially a bitmap-based drawing surface where you can dynamically create and manipulate graphics, animations, and interactive elements.
Canvas graphics are rendered pixel by pixel, making it suitable for complex, dynamic visualizations and games.
Canvas does not retain the shapes drawn on it, so if you want to interact with the individual elements later, you have to manage that manually in JavaScript.
Example use cases include real-time data visualization, gaming, photo editing applications, and more.

2.SVG:

SVG stands for Scalable Vector Graphics, which is an XML-based language for describing 2D graphics.
Unlike Canvas, SVG is resolution-independent, meaning graphics can be scaled without losing quality.
SVG graphics are composed of shapes and paths defined by mathematical equations, making them suitable for static or less interactive graphics.
SVG elements can be styled and animated using CSS and JavaScript.
Because SVG elements are retained in the DOM, they can be easily manipulated, animated, and interacted with using JavaScript or CSS.
Example use cases include logos, icons, charts, diagrams, and illustrations..
